# Tetris

### About

This project aims to build a modern Tetris game that can interact with Reinforcement Learning agents. It can be also played by human and supports features such as **hard** **drop**, **hold** **queue** and **T-spin**.

The project is currently work in progress.

### Actions

    0 : noop
    1 : move left
    2 : move right
    3 : move down
    4 : hard drop
    5 : rotate clockwise
    6 : rotate counter-clockwise
    7 : hold/dequeue
